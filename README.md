Chatbox API is a backend for the project [Chatbox][1]

Install and Run
------------
- Prerequisites: docker and docker-compose

Run the project

```
$ docker-compose up
```

API will listen at port 8000

Docker-Compose
------------
- Database: Mysql 5.7
- PHP-FPM
- Ngnix


[1]: https://github.com/diogojpina/chatbot-account-front
